# Cub

Fetch stock information

## Installation

Add this line to your application's Gemfile:

    gem 'cub'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install cub

## Usage

    Cub.company(7203)
    => "トヨタ自動車"

    Cub.price(7203)
    => 3275
